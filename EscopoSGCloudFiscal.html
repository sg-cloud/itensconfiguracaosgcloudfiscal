Ao emitir uma NF-e utilizando CST 20 ocorreu erro no Schema do XML por não haver a tag pRedBC (percentual da redução da base de cálculo). No tributador antigo não possui campo para preenchimento da informação.

Falha no Schema XML do lote de NFe. org.xml.sax.SAXParseException; lineNumber: 1; columnNumber: 1963; cvc-complex-type.2.4.a: Invalid content was found starting with element &apos;vBC&apos;. One of &apos;{"http://www.portalfiscal.inf.br/nfe":pRedBC}&apos; is expected.

O QUE FAZER:

Enviar a tag <pRedBC> no grupo <ICMS> do XML, quando o campo "Red. B.C ICMS" for preenchido no Grupo de Imposto.

> Trecho de um XML para exemplo:

<ICMS>-

<ICMS20>

<orig>6</orig>

<CST>20</CST>

<modBC>3</modBC>

<pRedBC>41.66</pRedBC>

<vBC>29.17</vBC>

<pICMS>18.00</pICMS>

<vICMS>5.25</vICMS>

</ICMS20>

</ICMS>

- Cálculo do ICMS com redução: Base de Cálculo cheia - Percentual Redução da B.C% * Percentual ICMS% = Valor do ICMS com Redução.
