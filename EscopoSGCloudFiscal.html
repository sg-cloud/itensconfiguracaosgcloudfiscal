#1412656 MELHORIA NO CONTABILISTA, PLANO DE CONTAS E VINCULOS DE PLANO DE CONTA S COM CFOP SGCLOUD carolinaferreira Tarefa 09/08/24 assigned 13.0 1

ONDE:

SGCloud - Editar Organização - Contabilista

O QUE OCORRE:

Para geração de arquivos fiscais algumas alterações são necessárias dentro do sistema, será necessário um plano de contas novo, adições de informações para o contabilista e melhoria dos CFOPs.

O QUE FAZER:

Requisito 1

Necessário criar uma mascara para plano de contas do contabilista, para isso serão necessários 27 Caracteres disponíveis.

- Adicionar no cadastro de filial, na aba de contabilista, o input de &apos;mascara do plano de contas&apos;

> Tratar para que enquanto nao existir registro no plano de contas contabil, ser possível alterar a mesma, uma vez criado um plano de contas a ser criado no REQ02, o mesmo não poderá sofrer alteração;

Requisito 02

Criar submódulo para gerar &apos;Plano de contas contábil&apos;.

Deverá ser necessário ter configurado a máscara no requisito 1 para criação/alteração, se nao tiver sido criada deverá apresentar notificação ao usuário: &apos;Atenção! Para ser criado um plano de contas contábil, primeiro deve ser criado a máscara do contabilista nos parâmetros do editar organização&apos;

Filtros da listagem: Código, Código conta, Descrição, Tipo e Natureza.

Colunas: (Código (gerado sequencial sistema), Código conta (seguir máscara), Descrição , Tipo (analítica/sintética), Natureza (opções de: Contas de ativo, Contas de passivo, Patrimonio liquido, Contas de resultado, Contas de compensação, Outros), Nível, Data inclusão/alteração.

- Coluna código: Abrir vazio, podendo ser preenchido pelo usuário (validar para nao duplicar), se salvar sem preencher deverá pegar o menor código disponível, seguindo a sequencia,

- Coluna código conta: Deverá ser preenchido pelo usuário, seguindo a definição da máscara definica no cadastro do contabilista.

- Coluna descrição: Input texto.

- Coluna tipo: select entre analitica ou sintétita.

- Coluna natureza: select entre Contas de ativo, Contas de passivo, Patrimonio liquido, Contas de resultado, Contas de compensação, Outros.

- Coluna nível: input desabilitado (será preenchido de forma automática EX: masca #.#.##.##.######.######

plano 1.1.11.111111 será o nível 5, ex2: 1.1.11 será nível 3.

- Coluna Data: input desabilitado, apresentando sempre a data da última alteração de dados.

Requisito 3

Necessário criar opção na tabela de CFOP. Criar um vinculo entre a tabela de CFOP com uma das contas do plano. Essa informação é enviada para o EFD contribuições.

No editar de um CFOP, criar um input select (Código de conta contábil - EFD Contribuições) onde apresentará a lupa de pesquisa, ao pesquisar devirá abrir modal de pesquisa com os códigos contábeis cadastrados em tabela criada no requisito 2.

Parâmetro:

Nome: Código de conta contabil de depreciaçãoDescrição explicativa: "Este parâmetro definirá qual será o código de conta contabil atribuido para a depreciação".Valor: Vazio default (input select - pesquisa tabela de códigos contábeis (req 02) PIS/COFINS)Módulo: FISCALGrupo: SPED EFD CONTRIBUIÇÕES
