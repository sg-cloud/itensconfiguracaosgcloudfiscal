Criar marcação de RETIDO ou NÃO RETIDO para Pis e Cofins:

Caso retido, serão enviados os dados de pis e cofins no xml com a tag tpRetPisCofins = 1.

Caso não retido, serão enviados os dados de pis e cofins no xml com a tag tpRetPisCofins = 2.

Para IRRF, CSLL, e INSS, quando informados valores nesses campos, considerar como retido.
