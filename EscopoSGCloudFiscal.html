- ADICIONAR OPCAO DE DADOS ADICIONAIS NA GERACAO DO SPED

Sugestão: Abrir modal ao clicar no botão

P1:

- Apresentar "Dados do período anterior":

- Inputs desabilitado de:

- Valor credor de ICMS

- Valor credor de ICMS de substituição

- Valor credor de IPI

- Apresentar "Dados do período atual":

- Inputs de:

- Valor credor de ICMS

- Valor credor de ICMS de substituição

- Valor credor de IPI

- Botão "Transportar valores"

- Ele pega os dados apresentados do período anterior e aplica no período atual

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

P2:

- Criar 3 Abas de ajustes:

- Ajuste de Crédito

- Ajuste de Débito

- Ajuste de IPI Crédito/Débito

1 - Ajuste de Crédito (print auxilio ajuste de credito de icms)

- Input &apos;Código de Ajuste&apos;

- os códigos de ajuste possuem mascara padronizada (UFBCDDDD)

- onde: UF = (PR,SP,SC, etc)

- B = Apuração

- C = Utilização (para crédito C = só pode estar em (2, 3 e 4)

- DDDD = Sequencia

- Input &apos;Valor&apos;

2 - Ajuste de Débito (print auxilio ajuste de débito de icms)

- Input &apos;Código de Ajuste&apos;

- os códigos de ajuste possuem mascara padronizada (UFBCDDDD)

- onde: UF = (PR,SP,SC, etc)

- B = Apuração

- C = Utilização (para débito C = só pode estar em (1 e 5)

- DDDD = Sequencia

- Input &apos;Valor&apos;

3 - Ajuste de Crédito/Débito de IPI (prints Ajustes de IPI crédito/débito e indicação de documento para auxilio)

- Select &apos;Tipo de Ajuste&apos;

- opções de:

- 0 - Ajuste a débito

- 1 - Ajuste a crédito

- Input &apos;Valor (R$)&apos;

- Select &apos;Código do Ajuste&apos;:

- Se tipo de ajuste = 0 - Ajuste a Débito

- Selects disponíveis de:

- 101 - Estorno de Crédito

- 102 - Transferência de Crédito

- 103 - Ressarcimento/Compensação de Créditos de IPI

- 199 - Outros Débitos

- Se tipo de ajuste = 1 - Ajuste a Crédito

- Selects disponíveis de:

- 001 - Estorno de Débito

- 002 - Crédito Recebido por Transferência

- 010 - Crédito Presumido IPI Ressarcimento PIS/Cofins Lei 9.363/1996

- 011 - Crédito Presumido IPI Ressarcimento PIS/Cofins Lei 10.276/2001

- 012 - Crédito Presumido IPI Regiões Incentivadas Lei 9.826/1999

- 013 - Crédito Presumido IPI Frete MP 2.158/2001

- 019 - Crédito Presumido IPI Outros

- 098 - Créditos decorrentes de Medida Judicial

- 099 - Outros Créditos

- Select &apos;Indicação de Documento&apos;

- opções de:

- 0 - Processo Judicial

- 1 - Processo Administrativo

- 2 - PER/DCOMP

- 3 - Documento Fiscal

- 9 - Outros

- Input &apos;N° do documento&apos;

- Input &apos;Descrição Detalhada&apos;

- Botão de (+) para adicionar os ajustes e inserir preview do que está sendo adicionado igual prints demonstrativas se possível

- Botão de &apos;Salvar&apos; e &apos;Cancelar&apos;
